# 🦆 Goose TD

A dynamic AI-powered Tower Defense game built with Google AI Studio and Gemini.

## 🎮 Play Now

**[Launch Goose TD](https://ai.studio/apps/195f3ffd-9acb-4858-aac1-8d6a741b6289)**

## About

Goose TD is an innovative tower defense game powered by Google's Gemini AI. Defend your territory against waves of enemies with strategic tower placement and real-time AI assistance.

## Features

- 🤖 AI-powered game dynamics
- 🛡️ Strategic tower defense gameplay
- 🎯 Progressive difficulty levels
- 💾 Built with Google AI Studio
- 🚀 Instant web deployment

## Getting Started

1. Visit the [live game](https://ai.studio/apps/195f3ffd-9acb-4858-aac1-8d6a741b6289)
2. No installation required - play in your browser
3. Check out [guides](./docs/) for strategies and tips
